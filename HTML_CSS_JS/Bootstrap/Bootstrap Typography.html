TYPOGRAPHY USING BOOTSTRAP

The following are the tags that uses bootstrap:

1.) .lead Makes a paragraph stand out

2.) .small Indicates smaller text (set to 85% of the size of the parent)

3.).text-left Indicates left-aligned text

4.) .text-center Indicates center-aligned tex

5.) .text-right Indicates right-aligned text

6.) .text-nowrap Indicates no wrap text

7.) .text-lowercase Indicates lowercased text

8.) .text-uppercase Indicates uppercased text

9.) .text-capitalize Indicates capitalized text

10.) INITIALISM Displays the text inside an abbr element in a slightly smaller font size

  • 11.) .list-unstyled
    • Removes the default list-style
    • Left margin on list items
  • 12.) .list-inline
  • Places all list items on a single line
    • 13.)
    .dl-horizontal Lines up the terms (dt) and descriptions (dd) in dl elements side-by-side. Starts off like default dl, but when the browser window expands, it will line up side-by-side
    14.) .pre-scrollable Makes a pre element scrollable
